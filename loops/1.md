* In terraform loops uses only when building maps or listing comprehension expressions.

```hcl
locals {
  double        = [for num in var.numbers1 : num * 2]
  even          = [for num in var.numbers1 : num if num % 2 == 0]
  odd           = [for num in var.numbers1 : num if num % 2 != 0]
  fname_list    = [for persion in var.persion : persion.fname]
  lname_list    = [for persion in var.persion : persion.lname]
  keys          = [for key, value in var.map_list1 : key]
  values        = [for key, value in var.map_list1 : value]
  values_double = [for key, value in var.map_list1 : value * 2]

  # Creating a map object from existing map object
  double_map = { for key, value in var.map_list1 : key => value * 2 }
}
```

* For loop can be used with if else statement also
